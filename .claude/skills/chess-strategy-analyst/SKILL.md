# Chess Strategy Analyst Skill

Этот скилл запускает агента `chess-strategy-analyst` для глубокого анализа шахматного качества игры LLM-моделей.

## Когда использовать

- После прогона новых партий — оценить как модели играли
- Для сравнения стратегии разных моделей
- Для выявления паттернов ошибок перед оптимизацией промптов
- После изменения промптов — оценить изменилось ли шахматное качество
- Когда нужно понять ПОЧЕМУ модели играют плохо (не просто метрики, а содержание ходов)

## Отличие от других агентов

| Агент | Что делает | Когда вызывать |
|-------|-----------|---------------|
| `/evaluate-models` | Рейтинг моделей по метрикам (composite, reliability, cost) | "Какая модель лучше?" |
| `/optimize-prompts` | Анализ промптов, рекомендации по тексту | "Как улучшить промпты?" |
| **`/analyze-strategy`** | **Глубокий шахматный анализ: как модели думают и играют** | **"Почему модели играют плохо?"** |

## 7 аспектов анализа

| # | Аспект | Что оценивается | Рубрика |
|---|--------|-----------------|---------|
| 1 | **Стратегия** | Оценка силы/слабости оппонентов, баланс агрессии | 0-5 |
| 2 | **Тактика** | Чтение позиции, расчёт вариантов, кандидаты | 0-5 |
| 3 | **Угрозы** | Распознавание входящих угроз, реакция | 0-5 |
| 4 | **Атаки** | Поиск захватов, шахов, тактических мотивов | 0-5 |
| 5 | **Дебют** | Развитие фигур, контроль центра, план | 0-5 |
| 6 | **Эндшпиль** | Реализация преимущества, выживание, промоушен | 0-5 |
| 7 | **Адаптация** | Смена стиля при выигрыше/проигрыше | 0-5 |

**Максимум:** 35 баллов. Идеальная модель не существует — текущий рекорд ~15-18.

## Режимы работы

### Полный отчёт (по умолчанию)
```
/analyze-strategy
```
Анализирует топ-6 моделей по всем 7 аспектам. 30-50 ходов на модель.

### Анализ одной модели
```
/analyze-strategy deepseek/deepseek-v3.2
```
Глубокий анализ одной модели, 50-100 ходов, 5-8 примеров на аспект.

### Фокус на аспекте
```
/analyze-strategy дебют
/analyze-strategy угрозы
/analyze-strategy эндшпиль
```
Один аспект по всем моделям.

### Разбор игры
```
/analyze-strategy game_01bc71cd-918a-4296-adb5-1ff723026007
```
Ход-за-ходом разбор конкретной игры.

## Данные

Агент читает:
```
logs/evaluations/model_rankings.json  — рейтинг с SmartBot scores
logs/evaluations/metrics.json         — per-move метрики
logs/game_<id>__<model>/move_NNN.json — трейсы ходов (thinking, board, moves)
prompts/*.txt                         — текущие промпты
```

## Результаты

Агент сохраняет:
```
logs/evaluations/strategy_analysis.json
```

Содержит: оценки по аспектам, ключевые находки, рекомендации для промптов.

## Pipeline анализа

Рекомендуемый порядок после прогона новых игр:

```
1. python -m trace_analyzer.metrics --smartbot  →  обновить метрики
2. /evaluate-models                              →  рейтинг моделей
3. /analyze-strategy                             →  шахматный анализ
4. /optimize-prompts                             →  оптимизация промптов
```

Каждый последующий шаг использует результаты предыдущего.
